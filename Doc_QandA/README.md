# 📄 PDF Q&A Bot (RAG-based Chatbot using Groq + FAISS + Hugging Face Embeddings)

A lightweight **Document Question Answering system** that allows you to upload PDFs and ask questions using natural language.

It uses **Retrieval-Augmented Generation (RAG)** with FAISS for vector search and **Hugging Face embeddings (no API key required)**.

---

## 🚀 Features

- 📂 Upload and process multiple PDF documents
- 🔍 Semantic search using FAISS vector database
- 🧠 Context-aware answers using Groq LLM (Llama3)
- 🤗 Hugging Face embeddings (free, no API key needed)
- ⚡ Fast local retrieval system
- 💬 Streamlit UI for interaction

---

## 🏗️ Tech Stack

- **Frontend:** Streamlit  
- **LLM:** Groq (Llama3 8B / 70B)  
- **Embeddings:** Hugging Face Transformers  
- **Vector DB:** FAISS  
- **Framework:** LangChain  
- **PDF Processing:** PyPDFDirectoryLoader
